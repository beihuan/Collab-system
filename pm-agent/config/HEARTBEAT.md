# PM Agent Heartbeat

> 每30分钟执行一次检查

## 检查项

### 1. 阻塞报告检查
- 读取飞书群组最新消息
- 如果发现新的阻塞报告（🚫标记），立即：
  - 分析该阻塞对依赖任务的影响范围
  - 评估是否需要调整其他人的任务安排
  - 通知人类并建议应对方案

### 2. 进度报告提醒
- 当前时间 > 19:30 时
- 检查多维表格中"进行中"的任务
- 对比飞书群组中的进度报告
- 如果某人未提交报告，发送提醒：
  ```
  ⏰ 提醒：{PERSON_NAME} 尚未提交今日进度报告，请尽快提交。
  ```

### 3. 逾期任务检查
- 查询多维表格中 due_date < 今天 且 status != "已完成" 的任务
- 如果发现逾期任务，通知人类：
  ```
  ⚠️ 逾期任务提醒：
  - [TASK-XXXX] {TITLE} — 负责人: {PERSON} — 原截止: {DATE}
  ```

### 4. 会议纪要处理
- 检查是否有人类提供的会议纪要待处理
- 如果有，触发 pm-meeting-assistant 的会后处理流程

### 5. 项目上下文更新
- 检查今天的苏格拉底式追问是否有新的结论
- 如果有，提醒人类确认是否更新项目上下文文档

## 异常处理

- 如果飞书CLI连接失败，记录错误并在下次心跳重试
- 如果多维表格读取失败，仅跳过相关检查项
- 连续3次心跳失败时，在终端显示警告
