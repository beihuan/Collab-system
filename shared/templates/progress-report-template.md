# 进度报告模板

```markdown
📊 进度汇报 — {PERSON_NAME} — {DATE}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 当前任务

- [TASK-XXXX] {任务标题} — 进度: {百分比}%
  - 状态: {进行中/待评审/已阻塞}
  - 今日进展: {具体做了什么}
  - 关联提交: {commit hash 或 PR 链接}

## 本周期已完成事项

- [TASK-XXXX] {任务标题} — ✅ 已完成
  - 产出: {具体产出物，如PR链接、文档链接}
  - 实际工时: {X}小时 (预估: {Y}小时)

## 下周期计划事项

- [TASK-XXXX] {任务标题} — 计划进度: {目标百分比}%
  - 具体计划: {明天/下个周期要做什么}

## 风险/卡点

- 🚫 [TASK-XXXX] {阻塞描述}
  - 原因: {具体原因}
  - 影响: {对其他任务的影响}
  - 需要协助: {需要谁提供什么帮助}

## 预计完成时间变更

- [TASK-XXXX] 原计划 {原日期} → 预计 {新日期}
  - 原因: {延迟原因}

## 代码活动

- 提交: {N}个commits
- PR: {PR链接和状态}
- 代码审查: {审查了哪些PR}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

> 此报告由 {PERSON_NAME} 的 Dev Agent 辅助生成，经人类确认后发送
```
